Introdução ao GitHub e Git

O que é GitHub?

O GitHub é uma plataforma online para hospedagem e gerenciamento de repositórios de projetos, permitindo colaboração entre desenvolvedores e controle de versões.

O que é Git?

O Git é uma ferramenta de controle de versionamento de código, que registra e gerencia o histórico de alterações de um projeto.

Main e Branch no GitHub

No GitHub, existe uma linha do tempo organizada em main e branch. O main é a ramificação principal do projeto, onde ficam as funcionalidades estáveis. O branch é utilizado para criar novas funcionalidades ou realizar alterações sem afetar o código principal.

Principais Comandos do Git

  • commit – Registra as alterações no repositório.
  • push – Envia novas alterações para o repositório remoto.
  • pull – Baixa atualizações do repositório remoto para o projeto local.
  • remote – Define ou gerencia a origem do repositório remoto.
  • add – Seleciona arquivos para serem incluídos no próximo commit.
